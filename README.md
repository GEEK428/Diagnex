# Diagnex - Clinical Search & Intelligence Platform

Diagnex is a high-precision clinical search engine designed for clinicians to navigate and map medical terminologies (ICD-11, NAMASTE, etc.) with the help of advanced AI intelligence.

## 📁 Repository Structure

- **Frontend**: React-based clinical interface with real-time AI mapping and search.
- **Backend**: Node.js/Express API handling clinical data, audit trails, and admin controls.
- **ML**: Python/Flask prediction service powered by a trained NLP model for medical concept extraction.

## 🚀 Key Features

- **Clinical Search Intelligence**: Optimized search across multiple terminology systems.
- **AI Precision Threshold**: Draggable confidence filtering (45%-100%) for diagnostic reliability.
- **Feedback Loop**: Expert clinician corrections for continuous model retraining.
- **Concept Lifecycle Management**: Admin tools for managing new medical terms and updates.
- **Premium Aesthetics**: Sophisticated, professional clinical theme for high-stress environments.

## 🛠 Tech Stack

- **Frontend**: React, Lucide, TailwindCSS (for specific components), Axios.
- **Backend**: Node.js, Prisma, PostgreSQL (Metadata), MongoDB (Audit/Feedback Logs).
- **AI/ML**: Python, Flask, Scikit-learn (Joblib), NLTK.
